Book Rental

Lease or rent a book

Technologies used: PHP, CodeIgniter, MySQL, Apache, jQuery, AJAX, Bootstrap.

A complete, full LAMP Stack book rental website for anyone who wants to rent a book or lease a book to others. Users can register to create an account to the system and adds book information to start leasing a book. User information is stored in MySQL database, and password is encrypted to ensure user safety. They can monitor who rented their books as well as update existing books or add a new book to their account. As a renter, user can choose from various categories and place it in the shopping cart then complete the purchase. Payment is validated by Stripe API. User information is validated in the back-end.
